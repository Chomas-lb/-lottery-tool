■抽選名と座席番号設定画面
お好きな名前と座席番号を最大値で設定してください。
50名の座席が販売していたら1～50。途中で不在の席が合っても問題ありません。

<img width="1339" height="813" alt="img_20251225(1)" src="https://github.com/user-attachments/assets/7c93e0e1-990e-4fbf-b043-9dc05fa24f23" />
■賞品登録画面
直接入力での追加と外部ファイル（テキスト）からの読み込みが可能です。
外部ファイルは改行して賞品を複数登録出来ます。

例
特等/温泉旅行
2等/遊園地ペアチケット

（/で１つの賞品名が2行に表示されます。/がない場合は1行で表示）
